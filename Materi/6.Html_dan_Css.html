HTML DAN CSS

HTML

HTML atau Hypertext Markup Language adalah sebuah markup language yang didesain untuk membuat halaman web atau informasi lain yang dapat dilihat oleh browser.HTML bukan sebuah BAHASA PEMOGRAMAN. Artinya HTML tidak punya kemampuan untuk membuat fungsionalitas yang dinamis. HTML juga dapat dibantu oleh teknologi seperti CSS (Cascading Style Sheets) dan Bahasa scripting seperti Javascript dan VBscript. jadi, HTML itu adalah bahasa mark up sangatlah mudah untuk dipahami bahkan bagi webmaster pemula di bi web development. Ada beberapa aplikasi yang didapat digunakan untuk membuat web menggunakan HTML seperti, Visual Studio Code, Sublime, dan Notepad++.

HTML adalah file yang diakhiri atau menggunakan ekstensi .html atau .htm. ekstensi file ini bisa dilihat dengan menggunakan web browser apapun (seperti Google Chrome, Safari, dan Mozilla Firefox). HTML memiliki fungsi untuk mengelola serangkaian data dan informasi sehingga suatu dokumen dapat diakses dan ditampilkan diinternet melalui layanan web.

Fungsi HTML yang lebih spesifik yaitu:

    1. Membuat halaman web

    2. Menampilkan berbagai informasi didalam sebuah browser internet

    3. Membuat link menuju halaman web lain dengan kode tertentu (hypertext)

SEJARAH VERSI HTML

1. 24 November 1995 : HTML 2.0 dipublikasikan sebagai IETF RFC 1866. Penambahan RFC memperbanyak kemampuan untuk:

    a) November 25, 1995: RFC 1867 (mengunggah file berdasarkan bentuk)

    b) Mei 1996: RFC 1942 (tabel)

    c) Agustus 1996: RFC 1980 (peta gambar berbasis klien)

    d) Januari 1997: RFC 2070 (internasional)

2. 14 Januari 1997 : HTML 3.2 dipublikasikan sebagai konsorsium world wide web. Ini adalah versi pertama yang dikembangkan dan di standardisasi secara khusus oleh konsorsium world wide web.

3. 18 December 1997 : HTML 4.0 dipublikasikan sebagai rekomendasi W3C. versi ini memiliki 3 variasi, yaitu:

    a) Ketat,di mana elemen terdeprekasi di larang.

    b) Tradisional, di mana elemen terdeprekasi diperbolehkan.

    c) Frameset, di mana sebagian besar hanya elemen yang berkaitan dengan frame diperbolehkan.

4. 28 Oktober 2014 : HTML versi 5.0 adalah versi terbaru yang sudah ada sejak tahun 2009 yang ditandatangani oleh organisasi IETF (internet Engineering Task Force) dan W3C (World Wide Web Consortium) dengan adanya tamabahan memiliki prosedur dalam pembuatan tampilan pada website baru melalui gabungan dari HTML dengan Javascript dan CSS.

CSS

CSS adalah singkatan dari Cascading Style Sheets.Secara konteks CSS atau Cascading Style Sheets adalah kumpulan perintah yang digunakan untuk menjelaskan tampilan sebuah halaman situs web dalam mark-up language. Fungsi CSS adalah untuk mendesain, membentuk, sertta mengubah tampilan halaman sebuah website.Dengan adanya CSS, tag HTML yang sederhana dapat diubah sehingga tampilan laman website menjadi terlihat menarik dan efisien.

MACAM-MACAM CSS

Ada tiga macam CSS yang sering digunakan web designer, yaitu:

    1. Inline Style Sheets, adalah CSS dengan perintah pemograman yang letaknya ada pada objek.

    2. External Style Sheet, CSS ini letaknya berbeda dengan laman yang akan diubah.CSS ini lebih praktis daripada Inline Style Sheets.

    3. Embedded Style Sheets, CSS model ini sama seperti Inline Style Sheets, sama-sama berada pada satu laman coding.

Hubungan CSS dan HTML

CSS dan HTML sangat berhubungan erat. CSS adalah kode-kode yang dipakai untuk mendesain sebuah halaman HTML. CSS akan membantu para web designer untuk mengubah tampilan teks (baik dari bentuk dan ukuran font maupun warnanya), menambahkan gambar, dan mengubah latar belakang sebuah laman HTML. Keberadaan CSS bisa terlihat dengan adanya atribut warna teks, gambar, latar belakang.
