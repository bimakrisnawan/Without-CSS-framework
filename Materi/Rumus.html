Rumus Microsoft Excel

Berikut merupakan rumus rumus yang "umum" digunakan dalam penggunaan Microsoft Excel:

    1. SUM = Digunakan untuk menjumlahkan nilai variable pada range tertentu.

    2. AVERAGE = Digunakan untuk mencari rata-rata pada suatu range data

    3. MAX = Digunakan untuk mencari nilai variable terbesar pada suatu range data.

    4. MIN = Digunakan untuk mencari nilai variable terkecil pada suatu range data.

    5. COUNT = Digunakan untuk menghitung banyak cell numeric pada suatu range data.

    6. COUNTA = Digunakan untuk menghitung banyak cell pada suatu range data.

    7. TRIM = Digunakan untuk menghilangkan spasi pada suatu cell. Meski sederhana, namun fungsi ini terbilang penting.

    8. IF(single) = Digunakan untuk mengolah data sesuai dengan ketentuan yang diinginkan.

    9. CONCATENATE = Digunakan untuk menggabungakan beberapa string menjadi 1 string.

    10. VLOOKUP = Digunakan untuk mencari informasi pada suatu range data yang beroperasi dari kiri ke kanan.

    11. HLOOKUP = Digunakan untuk mencari informasi pada suatu range data yang beroperasi dar atas ke bawah.

    12. FORMULA(AND) = Menghasilkan TRUE jika semua argumen yang diuji tersebut bernilai BENAR dan FALSE jika semua atau salah satu argumen bernilai SALAH.

    13. FORMULA(NOT) = Menghasilkan TRUE jika kondisi yang diuji tersebut bernilai SALAH dan FALSE jika yang diuji BENAR.

    14. FORMULA(OR) = Menghasilkan TRUE jika beberapa argumen bernilai BENAR dan FALSE jika semua argumen SALAH
